## 地図仕様

| 項目 | 値 |
|:---|:---|

| ズームレベル | 16 |
| ピン | 店舗位置に1つ |
| インタラクション | タップでGoogle Mapsアプリ起動 |

|:---|:---|

## API連携

| タイミング | API | 説明 |
|:---|:---|:---|
| 画面表示時 | GET /records/{recordId} | 記録詳細取得 |
| 削除時 | DELETE /records/{recordId} | 記録削除 |

## 共有機能

### 共有コンテンツ

| 項目 | 内容 |
|:---|:---|
| テキスト | 「{店舗名}に行きました！ ★{評価}」 |
| 画像 | メイン写真（1枚目） |
| URL | Google Mapsの店舗URL |

### 共有先

- LINE
- Twitter/X
- Instagram Stories
- その他（システム共有シート）

## 遷移先

| アクション | 遷移先 |
|:---|:---|
| 戻るタップ | 前画面（SCR-003 or SCR-004） |
| 編集タップ | SCR-005 記録編集画面 |
| 削除完了 | 前画面 |
| Google Mapsで開く | 外部アプリ（Google Maps） |
