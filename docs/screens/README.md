## 認証方針

- **初回起動時**: 匿名ユーザーIDを自動生成し、Keychainに保存
- **ログイン機能**: MVP後に追加予定（クロスデバイス同期、フレンド機能用）
- **データ保存**: 匿名ユーザーIDに紐づけてクラウド保存

## 共通UI要素

### ナビゲーションバー
- 戻るボタン（左）
- 画面タイトル（中央）
- アクションボタン（右）

### タブバー
- リストアイコン + "ホーム"（デフォルト選択）
- 地図アイコン + "マップ"

### FAB（フローティングアクションボタン）
- 画面右下に配置
- 新規記録作成へ遷移
- 全タブ共通で表示
